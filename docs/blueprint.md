# **App Name**: Academy Heroes

## Core Features:

- Customizable Dashboard: Student dashboard with avatar and background selection.
- XP and Gold Tracking: Display of experience points and gold.
- Secure Login: User Authentication: Secure login for students.
- AI Avatar Generator: Generate new avatar image options based on trending art styles. The tool will apply this style when reasonable.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent knowledge, trust, and the digital realm.
- Background color: A light, desaturated blue (#E1F5FE) for a clean and calming backdrop.
- Accent color: A lively green (#90EE90) to indicate progress, growth, and rewards.
- Body and headline font: 'Inter' (sans-serif) for a modern and easily readable interface.
- Use clear, simple icons to represent game elements and actions.
- Clean and intuitive dashboard layout for easy navigation and access to key information.
- Subtle animations to provide feedback on user interactions and add visual interest.
